다수공급자계약(MAS) 물품 산업안전보건관리비 계상방안 안내 — 요약

MAS 물품 계약금액에는 산업안전보건관리비가 별도 계상되어 있지 않으므로, 수요기관이 대상 물품에 대해 직접 산정·명시 후 납품요구하고 사후 실비정산 (설계내역으로 미리 계상한 경우는 그에 따름)  |  문의: 구매총괄과 042-724-7293·7273

1. 적용범위

  • MAS 계약서상 인도조건이 현장설치도이거나 설치비를 옵션으로 계약한 건
  • 물품설치가 산업안전보건법 제2조제11호 건설공사에 해당
  • 입찰공고문상 건설산업기본법·전기공사업법·정보통신공사업법·소방시설공사업법·국가유산수리법에 따른 공사면허 반영 (붙임 대상물품 참조)

2. 적용기준

  • 세부품명별·업체별 수요기관 납품요구금액(제품비＋설치비)
  • 2천만원 이상인 경우 계상 대상

4. 사후정산 방식

  • 산정 안내 납품요구서(기타사항)에 산업안전보건관리비 금액과 사후정산 가능 여부를 명시 후 업체 송신
  • 종결 시 지급 명시한 금액 이내로 실제 사용액을 추가 지급 (부가가치세 별도 지급)
  • 정산 기준 고용부 고시「건설업 산업안전보건관리비 계상 및 사용기준」제7조 사용기준에 따른 사용내역 확인 후, 부가세 제외 금액으로 정산 (해설집 p.40 참조)

5. 계상 예시

① 대상액 5억 미만·비완제품 — 미끄럼방지포장재(현장설치도), 납품 5,000만원 → (5,000만×70%)×3.11% = 1,088,500원
② 대상액 5억 미만·완제품 — 항온항습기, 총 4억(완제품비 3억) → ⓐ (4억×70%)×3.11%=8,708,000 / ⓑ [(4억−3억)×70%×3.11%]×1.2=2,612,400 → 작은 값 2,612,400원
③ 대상액 5억 이상·비완제품 — 미끄럼방지포장재, 납품 8억 → (8억×70%)×2.28% + 4,325,000 = 17,093,000원
④ 대상액 5억 이상·완제품 — 항온항습기, 총 20억(완제품비 12억) → ⓐ 36,245,000 / ⓑ 20,511,600 → 작은 값 20,511,600원
